##  Descripción

En el área de las Ciencias de la Computación, un compilador es una herramienta de software encargada de analizar y procesar programas escritos en un lenguaje de programación, verificando que cumplan las reglas léxicas, sintácticas y semánticas definidas.

El presente proyecto consiste en el desarrollo de un compilador básico para un lenguaje de programación diseñado mediante una gramática formal inspirada en C++. Este lenguaje incluye:

- Declaraciones de variables
- Asignaciones
- Operaciones aritméticas
- Entrada y salida (cin, cout)
- Estructuras condicionales (if-else)
- Ciclos (while, for)
- Funciones y sentencias return

La especificación del lenguaje se realiza mediante gramáticas formales utilizando notaciones BNF y EBNF, permitiendo describir de forma precisa su estructura sintáctica.

El compilador implementa las principales fases:

### Análisis Léxico
Encargado de reconocer los tokens del lenguaje como identificadores, palabras reservadas, operadores, números y cadenas.

### Análisis Sintáctico
Verifica que las instrucciones cumplan la estructura definida por la gramática.

### Análisis Semántico
Evalúa la coherencia y significado de las sentencias, como el uso correcto de variables.

### Manejo de Errores
Se incluye detección y reporte de errores en cada fase del proceso de compilación.

---

## Objetivo

Aplicar conceptos fundamentales de:
- Lenguajes formales
- Gramáticas
- Teoría de autómatas
- Diseño de compiladores

para construir un compilador funcional a nivel académico.
